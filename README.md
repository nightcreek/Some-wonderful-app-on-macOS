# Some-wonderful-app-on-macOS
## 一些好用的macOS软件推荐
因为macOS允许从App Store之外下载软件，所以有许多软件并不会上架App Store，这里我将列出几款好用的软件
### 包管理器
* #### 1.Homebrew
  * ##### 介绍
    这是一款独立运行的命令行工具，能够方便的管理应用程序
  * ##### 安装
    从[官方网站](https://brew.sh)获取下载或者复制以下代码将其拷贝到终端中
  ```

  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  ```
    * ###### 安装中可能出现的问题
    * 
  * ##### 使用
    Homebrew是一个命令行工具，所以它的基本使用是在终端中进行的，以下是Homebrew的基础命令
    *`brew install`最基础的下载指令，在后面加上你想要的app的英文名称就能下载了
    *例如：我要下载一个名叫IINA的app就可以使用以下命令`brew install iina`
           要卸载它则使用`brew install iina`
