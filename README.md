### 包管理器
* #### 1.Homebrew
  这是一款独立运行的命令行工具，能够方便的管理应用程序，可以从它给官方网站[https://brew.sh](https://brew.sh)获取下载或者复制以下代码将其拷贝到终端中
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
在中国大陆地区可以选择镜像源来保证下载的顺利进行
